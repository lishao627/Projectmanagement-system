# 项目管理系统（Project Management System）

## 一、项目简介

本项目是一个用于**学生学习任务与项目管理**的小型系统，主要面向高校学生在日常学习过程中对课程作业、实验报告等内容的整理与管理需求。

系统以“学生”为核心对象，先录入学生的基本信息，再在对应学生名下添加其大学期间的**作业、实验报告或课程项目**，从而实现对学习项目的集中管理与追踪。本项目为学校**认知实验课程**的小组实践项目，侧重于对信息组织、结构设计和基础编程能力的训练。

项目整体规模适中，逻辑清晰，适合教学演示与课程实验使用。

---

## 二、项目功能概述

当前版本主要实现以下功能：

* 学生基本信息管理

  * 添加学生基本信息（如姓名、学号、专业等）
  * 查询学生信息

* 学生项目 / 作业管理

  * 为指定学生添加大学作业或实验报告
  * 记录作业/实验的名称、类型及相关说明
  * 按学生维度管理其学习项目

* 项目管理逻辑

  * 通过“学生 → 项目”的结构，实现清晰的层级管理
  * 便于后续扩展评分、状态跟踪等功能

---

## 三、系统设计思路

本系统采用面向对象的设计思想，将现实中的管理关系抽象为程序中的对象关系：

* **学生（Student）**：作为核心实体，保存学生的基本信息
* **项目 / 作业（Project / Assignment）**：隶属于某一学生，用于描述其学习任务

通过组合关系，将学生与其对应的作业或实验报告进行关联，使系统结构更加贴近真实的学习场景，也便于理解与维护。

---

## 四、运行环境与技术说明

* 开发语言：Java（Java SE）
* 运行方式：控制台程序（命令行交互）
* 开发工具：IntelliJ IDEA / Eclipse（均可）
* 适用对象：大学课程实验、认知实验、小型课程设计

---

## 五、使用说明（简要）

1. 运行主程序
2. 按提示先添加学生基本信息
3. 进入学生信息管理模块
4. 为指定学生添加作业或实验报告
5. 根据需要查看或管理已有信息

系统采用菜单式交互，操作直观，便于上手。

## 六、项目意义

本项目通过一个贴近大学生日常学习的管理场景：

* 帮助理解**面向对象建模思想**
* 训练基本的数据组织与逻辑设计能力
* 提升对“项目管理”这一概念的实际认知

同时，也为后续更复杂系统（如带界面或数据库的管理系统）打下基础。

---

## 七、说明

本项目为课程实验用途，功能实现以教学目标为导向，未涉及复杂的异常处理和持久化存储。后续可根据学习进度逐步扩展相关功能。

如有不足之处，欢迎交流与指正。
